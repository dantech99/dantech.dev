Desarrollé y mantuve el sitio web para la empresa de gestión catastral de Colombia, enfocado en la publicación y administración eficiente de comunicados, informes y servicios para la ciudadanía.
Participé activamente como desarrollador principal, realizando más de 40 contribuciones directas en el código.

Logros destacados:

- Implementación de funcionalidades clave para la gestión de comunicados y servicios en línea.
- Optimización de la interfaz de usuario usando JavaScript, CSS y Blade.
- Integración de funcionalidades administrativas para facilitar la publicación de información oficial.
- Refactorización de módulos para mejorar la mantenibilidad y escalabilidad del proyecto.
- Apoyo en la integración de backend en PHP y frontend en tecnologías web modernas.
